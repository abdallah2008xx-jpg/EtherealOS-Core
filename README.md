# EtherealOS - Premium Gentoo Linux Distribution

EtherealOS is a high-end, visually stunning Linux distribution built on Gentoo with a focus on glassmorphism design, performance optimization, and user experience.

![EtherealOS Theme](bg.jpg)

## Quick Installation

### For New Users (First Time Install)

Open a terminal and run:

```bash
# Clone the repository
git clone https://github.com/abdallah2008xx-jpg/EtherealOS-Core.git ~/ethereal-update

# Run the installer
cd ~/ethereal-update
bash install-ethereal.sh
```

### For Existing Users (Update System)

Click the **Update EtherealOS** icon on your desktop, or run:

```bash
bash ~/ethereal-update/Ethereal-Update.sh
```

## Auto-Update System

EtherealOS includes an automatic update system. When updates are available:

1. You'll see a notification
2. Click "Update EtherealOS" icon on desktop
3. System will download and apply updates automatically
4. All your apps and settings are preserved

## Theme Components

- **Cinnamon Desktop Theme** - `cinnamon.css`
- **GTK3 Theme** - `gtk.css`

## System Requirements

- Gentoo Linux (base system)
- Cinnamon Desktop Environment
- 4GB RAM minimum (8GB recommended)
- Internet connection for updates

## Manual Commands

```bash
# Update system
bash ~/ethereal-update/Ethereal-Update.sh

# Install/update icons
bash ~/ethereal-update/install-ethereal.sh

# Fix Firefox
bash ~/ethereal-update/Ethereal-Firefox-Fix.sh

# Run optimizer
bash ~/ethereal-update/Ethereal-Optimizer.sh
```

## Troubleshooting

**Icons not showing?**
Run: `bash ~/ethereal-update/install-ethereal.sh`

**Firefox not working?**
Run: `bash ~/ethereal-update/Ethereal-Firefox-Fix.sh`

**System issues?**
Run: **Fix Browsers & System** from desktop
